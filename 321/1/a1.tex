$rev(\epsilon) = \epsilon$\\

\noindent Inductive Step:\\
For the string $xy$ where $|y|=1$ assume:\\
$rev(xy) = y\  rev(x)$\\

\noindent Show $rev(xy)=rev(y)\  rev(x)$ where $|y| = n+1$:\\
$rev(xy)=rev(yb)\  rev(x)$

$=b\  rev(y)\  rev(x)$\\

\noindent Thus, $rev(xy) = rev(y)\  rev(x)\  QED$
